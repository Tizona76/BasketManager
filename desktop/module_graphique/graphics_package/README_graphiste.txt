Graphics brief – Basketball management game (Pygame)
====================================================

Context
-------
I am developing a basketball club management game (similar to a "basketball manager").
The game is already functional, but I want to significantly improve the visuals
without changing the game structure or code.

You do NOT need access to the game code, only to the visuals described below.
The screenshots in the "screenshots" folder (to be added by me) are for reference only.

Note about screenshots
----------------------
At this stage, I do NOT have proper screenshots yet for:
- The in-game match screen
- The "Pre-PO" (pre–Play-Off / decisive match) screen

Precisely for these two, I am looking for creative proposals from you, while still
respecting the current visual identity of the game (colors, general mood) as shown
in the available screenshots (home screen, season menu, etc.).

Target resolution
-----------------
Target screen format: 1280 x 720 px (16:9).
Please work in 1280x720 or in vector format that can be exported to this size.

High-value visuals to produce
-----------------------------

1) Main background – "Key Art Basket"
   - Size: 1280 x 720 px.
   - Format: PNG, no text.
   - Content:
     * Modern basketball court atmosphere, seen from 3/4 or from the side/front,
       with stands and spotlights (professional game feeling).
     * A relatively neutral / darker central area so I can place the logo
       and buttons on top and keep them readable.
   - Style:
     * Modern, serious (not childish/cartoon),
     * Lights, depth, slight glow,
     * Coherent with a management game (professional atmosphere, not arcade neon chaos).

2) Game logo
   - Format: PNG with transparent background.
   - Versions:
     * Large horizontal version (~800x250 px max) for the main menu / home screen.
     * Compact version (~300x300 px) for other screens.
   - Content:
     * Game name: "Basket Manager" (or leave room so I can adjust the exact text).
     * Integration of a basketball, shield/crest, or strong graphical element.
   - Style:
     * Inspired by NBA/Euroleague style logos (without copying),
     * Colors coherent with the main background (e.g. dark blue, basketball orange, white).

3) Main UI button pack
   - Format: PNG, no text (button labels will be rendered by the game).
   - Target size: about 400 x 80 px.
   - 3 states for the same button design:
     * Normal
     * Hover (slightly brighter, subtle glow or outline)
     * Pressed / clicked (slightly darker, "pressed in" effect)
   - Style:
     * Strongly rounded rectangles (pill shape),
     * Light gradient or volume effect,
     * Consistent with the logo (same color language).

4) "Score / Match result" panel
   - Format: PNG.
   - Indicative size: ~900 x 400 px, displayed in the center of the screen.
   - Graphic content:
     * A "TV scoreboard" style panel where I will display (via text in the game):
       - Home team name
       - Opponent team name
       - Final score
       - "Victory" / "Defeat" label
     * Please DO NOT include fixed text: only the graphic frame/background.
   - Style:
     * Consistent with the main background,
     * Clean and dynamic design (borders, frames, small decorative elements),
     * Very readable.

5) "Pre-PO" (pre–Play-Off / decisive match) visual
   - Format: PNG.
   - Size: either full screen 1280x720 OR a big central panel similar to the result panel (~900x400).
   - Graphic content:
     * More dramatic atmosphere: stronger contrast, spotlights, feeling of a crucial game.
     * A central zone where I can display (via text in the game):
       - "Pre-PO" or "Decisive match"
       - Match pairing (My team vs Opponent)
     * Please DO NOT include fixed text.
   - Style:
     * A more intense variation of the main background,
     * Must clearly stand out visually from a normal match screen,
     * Free creative proposal welcomed, as long as it remains consistent
       with the general visual identity of the game.

Deliverables
------------
- All files in PNG (and source files if possible: PSD/AI/FIGMA/SVG, etc.).
- Color palette used (optional but appreciated).
- If possible, a simple mockup showing how you imagine:
  - background + logo + buttons together on a home screen.

Important
---------
- Please do NOT hardcode text (except inside the logo itself if necessary):
  I want to keep full control over all texts inside the game.
- I prefer clean, readable visuals over too many effects.
- The style should feel like a modern sports management game
  (not childish, not ultra-arcade).
